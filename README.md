# KeenKeeper – Friendship Analytics & Interaction Tracker

KeenKeeper is a modern React-based interaction tracking and analytics web application. It helps users record daily interactions (calls, messages, video calls) and visualize activity patterns through an interactive dashboard.

---

## Live Concept

A personal engagement tracker that transforms everyday interactions into meaningful insights using charts and timeline history.

---

## Key Features

### Interaction Tracking System
Users can log activities in real time such as Audio Calls, Messages, and Video Calls. Each activity is stored with type, user name, and timestamp.

---

### Timeline
Chronological activity history built using React Context API with persistent storage via localStorage. Supports adding activities and automatic sync updates.

---

### Analytics Dashboard
Visual breakdown of user activity using Recharts (Pie Chart). Displays interaction distribution (Call vs Message vs Video). Fully responsive and dynamic.

---

### Filtering System
Timeline can be filtered by All Activities, Messages, Calls, and Videos with instant UI updates without page reload.

---

### User Feedback System
Integrated React Toastify for real-time notifications including call started, message sent, and video call started.

---

### Responsive UI
Mobile-first design with adaptive layout for all screen sizes, clean card-based interface, and optimized navigation experience.

---

## Tech Stack

React.js, React Router DOM, Context API, Tailwind CSS, Recharts, React Toastify

**KeenKeeper – Friendship Analytics & Interaction Tracker**

KeenKeeper is a modern React-based interaction tracking and analytics web application that helps users log and analyze their daily communications. It allows you to record interactions such as calls, messages, and video calls, and visualize activity patterns through an interactive dashboard.

**Key Features**
Interaction Tracking System

Users can log real-time activities including Audio Calls, Messages, and Video Calls.
Each entry is stored with:

1. Interaction type
2. User name
3. Timestamp

**Timeline**

A chronological activity feed built using React Context API with persistent storage via localStorage.

1. Add new interactions instantly
2. Auto-sync updates
3. Clean and readable activity history


**Analytics Dashboard**

An interactive dashboard powered by Recharts.

1. Pie chart visualization of interaction types
2. Breakdown of Calls, Messages, and Video Calls
3. Fully responsive and dynamic UI

**Filtering System**

Easily filter timeline activities by category:

1. All Activities
2. Messages
3. Calls
4. Video Calls

Instant updates without page reload for a smooth user experience.

**User Feedback System**

Integrated React Toastify for real-time notifications:

1. Call started
2. Message sent
3. Video call initiated

Provides instant feedback for every user action.

**Responsive UI**

A mobile-first design optimized for all screen sizes:

1. Adaptive layout
2. Clean card-based interface
3. Smooth navigation experience

**Tech Stack**
1. React.js
2. React Router DOM
3. Context API
4. Tailwind CSS
5. Recharts
6. React Toastify
7. LocalStorage API

**Purpose**

KeenKeeper is designed to demonstrate state management, data visualization, and real-time UI updates in a modern React application.
